## fastlive

RTMP server implementation for GO.  

Contributions and pull requests are warmly welcome! 



### Installing fastlive

1. Get fastlive

   ```
   git clone git@github.com:xly0713/fastlive.git /path/to/fastlive
   ```

2. Build fastlive

   ```
   cd /path/to/fastlive && make
   ```

3. Run fastlive

   ```
   ./bin/fastlive
   ```
